Al enviar una petición POST a la ruta /registro con el siguiente JSON:

{
    "nombre": "Maria",
    "mensaje": "Hola comunidad"
}

el servidor devuelve una respuesta en formato JSON confirmando que los datos fueron recibidos correctamente:

{
    "estado": "Datos recibidos",
    "nombre": "Maria",
    "mensaje": "Hola comunidad"
}
}
